Table Buddy Demonstration App.
  • Table from backend data with server-side paging
    Demonstrates use of Table Buddy via 2-stage backend API calls with in-page rendering of sorting, filtering, and paging controls.
  • Table from backend data with client-side paging
    Demonstrates use of Table Buddy via alternative 2-stage backend API calls whereby paging is handled client-side. Rendering of sorting, filtering, and paging controls is handled by in-page code.
  • Table from single backend data call
    Demonstrates use of Table Buddy via single backend API call that returns all data. Rendering of sorting, filtering, and paging controls is handled by in-page code.
  • Table from backend data with Filter Buddy component
    Demonstrates use of Table Buddy via 2-stage backend API calls whereby paging is handled client-side. Rendering of sorting is handled by in-page code. Rendering of filtering is handled by Filter Buddy and rendering of paging controls is handled by a sampl Web Component.
